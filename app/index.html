Help

Selecting tutorial
  • To chose tutorial click on the "Tutorials" button in the navigation. You will see a dialog box, inside its content you will be able to preview all different tutorials.
  • Each tutorial has name, description, level and tags. The easiest tutorials are with one star for their level, the most complex tutorials are with three stars.
  • Using the search bar you can filter the tutorials.
  • When you chose a tutorial just click on it and its first step will be opened.
Solving tutorial
To complete the tutorial read the required steps in the bottom-left corner, write JavaScript in the top-left corner and HTML in the top-right one. The result will be visualized in the bottom-right corner.
UI features
  • If you want to make any of the text boxes for HTML or JavaScript wider just click on its content.
  • In order to change the theme to dark use the black box, left from the "Help" button. To restore to the previous theme use the white box.


Have fun!
